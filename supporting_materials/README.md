# Supporting Materials

This folder contains aggregate-only support for numbers discussed in the rebuttal responses. It does not include prompt templates, participant-level rows, raw psychometrics, generated participant narratives, real conversation transcripts, or transcript-level annotations.

- `tables/`: small aggregate tables corresponding to the main checks.

The tables are intended to make the response numbers easier to audit without expanding the rebuttal into a full supplementary appendix. For the LSI recovery matrix, the public table is limited to the 24-section psychometric-only and interwoven-biography conditions.

Recent additions:

- `headline_24turn_lsi_recovery_by_generator_condition_scorer.tsv`: 24-section LSI recovery by headline generator, profile condition, and scorer.
- `headline_24turn_lsi_recovery_summary_by_generator_condition.tsv`: compact summary of the same matrix.
- `profile_writer_recovery_opus_qwen37_gemma.tsv`: aggregate profile-decoding strength for Opus, Qwen 3.7 Max, and Gemma 4 31B profile writers where full-290 scored runs were available.
