# COLM 2026 LSI Rebuttal Materials

This repository contains the COLM 2026 rebuttal response text for the Life Story Interview submission.

It is organised around the reviewer responses rather than the internal analysis history:

- `reviewer_responses/`: paste-ready responses, one per reviewer, with markdown headings and character counts.
- `GLOBAL_SUMMARY.md`: short cross-reviewer summary of the response strategy and evidence.
- `supplemental_notes/SUPPLEMENTAL_NOTES.md`: brief context for points that were compressed by the 5,000-character response limit.
- `supporting_materials/CAMERA_READY_ADDITIONS.md`: short list of targeted camera-ready additions referenced in the responses.
- `supporting_materials/tables/`: aggregate-only tables for the main rebuttal numbers.
- `synthetic_demo/`: fully fictional, inspectable mini-demo showing sparse-fact reverse scoring, composite target construction, and profile reverse scoring without participant data or production profile-writing prompts.

The intended use is narrow: provide the response text and a small amount of context for compressed claims, without adding a broader evidence appendix or participant-level material.
