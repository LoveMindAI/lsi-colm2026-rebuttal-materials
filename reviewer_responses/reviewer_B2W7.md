# Method Variance, Human Check, and Robustness

Thank you for the supportive reading and for the specificity of your questions. The additional checks below respond directly to the concerns you raised while keeping the submitted claim narrow.

**Shared-method variance.** You are right that this is the central methodological caveat. The pipeline is very LLM-dense, so we conducted controls that remove or vary specific LLM-mediated steps. Deleting every generated-narrative token that also appeared in the Stage 1 profile removed about 20-26% lexical overlap, but HEXACO recovery changed by only `-0.001` to `-0.020`. Deterministic scorecards made directly from source psychometrics, with no LLM profile writer, recovered HEXACO/Beyond-HEXACO at `0.654/0.515` for a compact scorecard and `0.728/0.584` for a glossary scorecard, both below the matched psychometric-only narrative profile at `0.761/0.618`. Embedding geometry also preserves trait-neighbourhood structure for psychometric narratives, while a biography/facts-only control is near zero. If accepted, we will present these as triangulation and keep shared-method variance explicit.

**Real-conversation signal.** You are right that `ICC = 0.483` limits how much weight the real-conversation comparison can bear. Generated-story features correlate with the same features coded from real conversations at mean `|r| = 0.147-0.180`. Participants' own HEXACO scores predict those coded conversation features at mean `|r| = 0.094`; even choosing the best HEXACO domain separately for each feature gives `0.181`. PARSEL partner-perceived HEXACO correlates with self-report at mean `r = 0.157`, and independent LLM scorers recover HEXACO from the same short conversations at `r = 0.179-0.221`. If accepted, we will describe this as modest behavioural triangulation, remove the metric-mixing Figure 3 from the main text, and state that richer conversation data are needed for stronger behavioural claims.

**Human check.** We ran the most direct constructive check available under the current scope. A new Prolific pilot analysed only AI-generated materials: brief snippets of profile information and generated LSI themes, not full transcripts or full narratives. Twenty-five native-English raters each completed 25 four-option matches, passing bot/LLM screening and attention checks. Accuracy was significantly above chance, `59.5%` versus `25%`, `t(24)=8.24`, one-tailed `p<1e-8`. The limited information shown to raters makes this a conservative human check.

**Benchmark framing.** The submitted pipeline starts from measured psychometric profiles, transforms them into life-story prompts, and tests whether that information survives a generated LSI task. Most prior text-to-personality benchmarks infer personality from natural text where personality is only one diffuse signal among topic, genre, audience, and context. If accepted, we will describe prior numbers as scale references, not direct competitors. Wright et al. report `r = 0.44-0.48` from richer self-narration; PARSEL is much shorter and less introspective.

**Model-family bias.** We audited Stage 1 directly. We split the submitted Opus-written immersive prompt into psychometric-only and psychometric-plus-biographical profile conditions, then ran a full-N prompt-writer bakeoff across frontier API writers and locally hostable open-weight writers. Gemma 4 31B was selected as the camera-ready default because it was strong and locally hostable; in the full-290 audit, Gemma modestly exceeded Opus on HEXACO, beyond-HEXACO, PPTS, and SVO recovery.

We also re-ran the LSI recovery matrix across four generator families, two profile formats, two LSI lengths, and five scorers, partly because some submitted endpoints had shifted or been deprecated. All 80 rows are complete at `N = 290`. Same-family generator/scorer rows are not higher than cross-family rows on HEXACO/Beyond-HEXACO recovery: `0.751/0.600` versus `0.760/0.610`. If accepted, we will include the matrix and profile-writer audit in the appendix. We will also add the missing PPTS citation and clarify PPTS polarity.

**Emotional reactivity.** We generated unconditioned 24-turn life stories with no personality profile, at least 50 per generator family, using the same section-level coding. Unconditioned between-story SDs are `0.070-0.088`, compared with `0.102-0.131` for conditioned lanes; Brown-Forsythe tests separate conditioned from unconditioned lanes in 27 of 32 pairwise comparisons. The effect also remains after controlling for average valence, is strongest for HEXACO Sentimentality, and is robust across four independent LLM coders and four generator families. If accepted, we will report this as a bounded robustness check on the existing emotional-reactivity result, scoped to the 24-turn protocol and valence spine.
